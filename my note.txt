VERSION CONTROL SYSTEM
Examples include Git(Github, Gitbat), Bitbucket,etc
Uses of VCS include:
1. Tracking codes changes.
2. Knowing who makes changes to code.
3. Collaboration.

GIT BASH

git log - to check our commits.
